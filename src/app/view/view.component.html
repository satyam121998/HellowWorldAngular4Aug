This is a demo project

This particiular angular project contains total of four components excluding the app component created through the ng command of create a component by just simply running ng generate component component-name
Below are the details about the components and the tests running on the components.

Componet HelloWorld

My HelloWorld Component just contains a drop down list with selector and this drop down is taken from prime ng

Componet Card

My Card Component just contains a button which is event driven and this button is taken from prime ng

Componet Cart

My Cart Component just contains a list and a button and only the button is taken from prime ng and is event driven.
The list is taking the data from the service.ts file and the test is running with the service.spec.ts file

Test Data

The test has been performed by using Karma which is using the framework of Jasmine. To perform the test I just ran the command of ng test.
Just to test I have written an add function in my app.component.ts and wriiten an unit case in app.component.spec.ts file.
